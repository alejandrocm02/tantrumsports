# TantrumSports

Tienda web estatica de zapatillas con catalogo, filtros, carrito y vista 360 generativa.

## Paginas

- `index.html`: tienda principal.
- `carrito.html`: carrito con unidades, tallas y resumen.

## Publicar en GitHub Pages

1. Sube estos archivos a un repositorio de GitHub.
2. En GitHub, ve a `Settings > Pages`.
3. Selecciona `Deploy from a branch`.
4. Usa la rama `main` y la carpeta `/root`.
